# Fashion Flesta Admin Panel

Admin panel for Fashion Flesta e-commerce platform.

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS
- React 19

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the admin panel.

## Features
- Dashboard with stats overview
- Product management
- Category management
- Order management
- User management
- Banner management
